Catastrophic Forgetting Visualizations

Catastrophic forgetting occurs when a neural network completely loses previously learned knowledge upon learning new tasks. These interactive visualizations demonstrate this phenomenon across different model architectures and scenarios.

Each visualization provides an interactive demonstration where you can train models on different datasets and observe how new learning interferes with previously acquired knowledge.

Linear Classifier Demonstration
A foundational demonstration using a simple linear classifier to show catastrophic forgetting in its most basic form. Perfect for understanding the core concept.
• Two-class classification • Real-time decision boundary visualization • Step-by-step training phases
